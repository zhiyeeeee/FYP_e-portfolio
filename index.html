My Final Year Project

Project Title

A Preference-Based Optimization Model for University Badminton Court Scheduling Using a Genetic Algorithm


Background

This study develops a preference-based optimization model using a Genetic Algorithm to maximize user satisfaction while minimizing booking conflicts in university badminton court scheduling.


Research Objectives

  • Maximize user preference satisfaction.
  • Identify objective functions and constraints.
  • Develop a Genetic Algorithm model.
  • Evaluate scheduling performance.
